The Internet

The Internet is made of computers that speak to each other. When you access a website your computer makes an HTTP request to a server(s), and they return the correct HTML document, which is displayed by your computers browser. Your computer and servers are pretty much the same except servers are optimized to store files.

HTML

HTML = hypter text markup language and consists of: text content (what you see), markup (what it looks like), references to other docs, and links to other pages. An anchor tag is for links, this is an example.

Inline vs. Block Tags

Inline tags include the tags for bold, italicize, anchor, image, br, strong, etc. and end in the same line. Blocktags create an invisible box that has height and width, and is for tags like the paragraph and div tags.
